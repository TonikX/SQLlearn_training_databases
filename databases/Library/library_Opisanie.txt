Основными таблицами базы данных являются:

1)books (книги). Таблица содержит информацию о книгах библиотеки: 
идентификатор книги(book_id (PK)); 
ФИО автора книги(author); 
название книги(name); 
жанр книги(genre);
категория издания: научное, учебное, литературно-художественное(category); 
издатель книги(publisher); 
год издания(pub_year); 
тип издания: иностранное, отечественное(pub_type).
 
2)instances (экземпляры). Таблица содержит информацию об экземплярах книг: 
идентификатор экземпляра(instance_id (PK));
идентификатор книги(book_id (FK));
статус экземпляра(status); 
идентификатор работника, который зарегистрировал экземпляр(worker_id(FK));
шифр экземпляра(cypher);
дата регистрации экземпляра(date_reg);
дата списания экземпляра(write_off_date).

3)workers (работники). Таблица содержит информацию о работниках библиотеки:
идентификатор работника(worker_id(PK)); 
ФИО работника(full_name);
должность работника(position);
стаж работника(work_experience);
дата рождения работника(birthday);
образование работника: высшее, среднее профессиональное, среднее(education).

4)readers (читатели). Таблица содержит информацию о читателях: 
идентификатор читателя(reader_id (PK)); 
ФИО читателя(full_name_r);
номер паспорта(passport);
номер телефона(phone); 
вид занятости(occupation);
ученая степень, если имеется(science_degree);
форма обучения: очная, заочная(education_mode);
номер группы, если это студент(groups);
статус читателя, если студент, то обучается/отчислен, если преподаватель, то работает/уволен(status).

5)readers_reg (регистрация читателей). Таблица содержит информацию о регистрации всех читателей: 
номер читательского билета(library_card (PK)); 
дата регистрации(date_reg); 
идентификатор работника, который зарегистрировал читателя(worker_id (FK));
идентификатор читателя(reader_id (FK)).
 
6)reading_rooms (читальные залы). Таблица содержит информацию о читальных залах библиотеки: 
номер зала(hall_number (PK)); 
вместимость залы(capacity); 
название зала(name);
тип зала: зал научной, учебной или литературно-художественной литературы(species).

7)take_book (взять книгу). Таблица содержит информацию о выдаче экземпляров читателям на руки: 
идентифицирующий номер операции выдачи(op_id (PK));
дата закрепления читателем книги(fixing_date); 
идентификатор работника, который выдал книгу(worker_id(FK)); 
идентификатор читателя, который взял книгу(reader_id (FK)); 
идентификатор экземпляра книги, выданного читателю(instance_id (FK)); 
дата сдачи читателем книги(return_date); 
формальная дата сдачи читателем книги, то-есть срок сдачи книги(f_return_date); 
штраф, если читатель вернул книгу позже формальной даты сдачи(fine);
номер зала, в котором была выдана книга(hall_number (FK)).

Описание таблиц:
1) books - книги
	book_id (PK) - идентификатор книги
	author - ФИО автора
	name - название книги
	genre - жанр
	category - категория издания
	publisher - издатель
	pub_year - год издания
	pub_type - тип издания

2) instances - экземпляры
	instance_id (PK) - идентификатор экземпляра
	book_id (FK) - индентификатор книги
	status - статус экземпляра
	worker_id(FK) - идентификатор работника
	cypher - шифр экземпляра
	date_reg - дата регистрации экземпляра
	write_off_date - дата списания экземпляра

3) workers - работники
	worker_id(PK) - идентификатор библиотекаря 
	full_name - ФИО библиотекаря
	position - должность библиотекаря
	work_experience - стаж работника
	birthday - дата рождения работника
	education - образование работника

4) readers - читатели
	reader_id (PK) - идентификатор читателя
	full_name_r - ФИО читателя
	passport - номер паспорта
	phone - номер телефона
	occupation - вид занятости
	science_degree - ученая степень
	education_mode - форма обучения
	groups - номер группы
	status - статус читателя
	
5) readers_reg - регистрация читателей
	library_card (PK) - номер читательского билета
	date_reg - дата регистрации
	worker_id (FK) - идентификатор работника
	reader_id (FK) - идентификатор читателя

6) reading_rooms - читальные залы
	hall_number (PK) - номер зала
	capacity - вместимость
	name - название зала
	species - тип зала

7) take_book - взять книгу
	op_id (PK) - идентифицирующий номер операции
	fixing_date - дата закрепления читателем книги
	worker_id(FK) - идентификатор работника
	reader_id (FK) - идентификатор читателя
	instance_id (FK) - идентификатор экземпляра
	return_date - дата сдачи читателем книги
	f_return_date - формальная дата сдачи читателем книги
	fine - штраф
	hall_number (FK) - номер зала, в котором была выдана книга
